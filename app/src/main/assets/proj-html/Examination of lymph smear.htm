Lymph Node Smears
    1. A suitable superficial lymph node is selected and palpated so that the site is fully known.
    2. The site is then cleaned with alcohol.
    3. Using a suitable clean needle and an empty clean hypodermic syringe with all air expelled from it, the gland is punctured.
    4. The plunger of the syringe is then drawn back, drawing some of the fluid from the lymph node into the syringe.
    5. The needle is withdrawn from the site, still attached to the syringe.
    6. Lymph "juice" is ejected onto a clean microscope slide.
    7. Smear are then made following the procedure for making a blood smear using a spreader slide.
